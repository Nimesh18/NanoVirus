To run the jar file from the cmd prompt:

1. Change the current directory to the folder which contains the jar file
E.g. If NanoVirusApp is the folder which contains the jar file and it is located on the desktop, then 
	cd C:\Users\Nim\Desktop\NanoVirusApp

2.Type in the command
java -jar NanoVirusApp.jar

3.Done.

-------------------------------------------------------------------------------------------------------------------
Output:
The output of the program will be on the console as well as a text file output.txt
 located in the same directory as the jar file
